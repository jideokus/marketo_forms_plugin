== Installation ==

1. Upload `tsn-marketo-forms.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin by going to Marketo form settings and enter your:
	-Marketo REST API Endpoint
	-Marketo REST Identity
	-Marketo Client ID (you need to have a developer user created and an API web service set up for this)
	-Marketo Client Secret (same as above)
	-Look up field to ensure that the plugin searches for and updates the right person
4. Create a Marketo form by clicking on Add new
5. Add as many fields and save
6. Go to the page or post you want to add the form to
7. Click on the TSN Marketo Forms button in the editor
8. Put in the details and save
Then you're good to go
